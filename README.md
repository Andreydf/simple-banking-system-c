## Sistema bancário simples em C

Este projeto é uma simulação de um sistema bancário simples desenvolvido em linguagem C, com execução via terminal.

O objetivo principal foi praticar conceitos fundamentais de programação estruturada, organização de código e modelagem de dados.

## 🚀 Funcionalidades

- Consulta de saldo
- Depósito com validação de entrada
- Saque com verificação de saldo
- Registro de histórico de transações
- Interface interativa baseada em menu

## 🧠 Conceitos aplicados

- Estruturas de decisão (`if/else`)
- Laços de repetição (`while` e `for`)
- Uso de `struct` para modelar a conta bancária
- Funções para organização e reutilização de código
- Manipulação de arrays para armazenar histórico
- Uso de ponteiros para modificar dados diretamente

## 🏗️ Estrutura do código

O sistema foi organizado em funções para separar responsabilidades:

- `menu()` → exibe as opções
- `verSaldo()` → mostra o saldo atual
- `depositar()` → adiciona valor ao saldo
- `sacar()` → realiza saque com validação
- `mostrarHistorico()` → lista transações

## 📌 Exemplo de uso

```bash
--- BANCO ---
1 - Ver saldo
2 - Depositar
3 - Sacar
4 - Ver historico
5 - Sair
```
