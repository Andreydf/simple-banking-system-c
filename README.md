# simple-banking-system-c
💻 Sistema Bancário Simples em C

Este é um projeto simples desenvolvido em linguagem C durante meus estudos de programação.

A ideia foi aplicar conceitos básicos em um sistema funcional, simulando operações de um banco no terminal.

🚀 Funcionalidades
Consultar saldo
Realizar depósitos
Realizar saques com validação
Visualizar histórico de transações
Menu interativo no terminal
🧠 O que eu aprendi
Uso de estruturas condicionais (if/else)
Estruturas de repetição (while e for)
Entrada e saída de dados (scanf e printf)
Uso de arrays para armazenar informações
Validação de dados para evitar erros
⚙️ Como funciona

O programa simula uma conta bancária simples onde o usuário pode:

Ver o saldo atual
Depositar valores
Sacar valores (com verificação de saldo)
Visualizar todas as transações realizadas
📌 Próximas melhorias
Organizar o código usando funções
Criar suporte para múltiplas contas
Implementar sistema de login/senha
Salvar dados em arquivos
▶️ Como executar

Compile o código usando um compilador C:

gcc banco.c -o banco
./banco

Ou utilize um compilador online como GDB Online.
